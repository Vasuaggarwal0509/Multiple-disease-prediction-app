Research Methodology

Evaluation framework, model architectures, and dataset descriptions.

Evaluation Protocol
Image Classification Tasks
  • Models: ResNet-50, VGG-16, MobileNet V3 (all pretrained on ImageNet)
  • Training: Transfer learning with fine-tuning on disease-specific datasets
  • Input Size: 224 x 224 x 3 (RGB)
  • Data Split: Train / Validation / Test (as provided by dataset or 80/10/10)
  • Optimizer: Adam with learning rate scheduling
  • Loss: Categorical cross-entropy (multi-class) / Binary cross-entropy (binary)
  • Augmentation: Random rotation, flip, zoom, brightness adjustment
Tabular Classification Tasks
  • Model: Support Vector Machine (linear kernel)
  • Preprocessing: StandardScaler normalization
  • Data Split: 80% train / 20% test (stratified)
  • Validation: Single train/test split (small dataset sizes)
Metrics Definitions
Accuracy

Proportion of correct predictions over total predictions. Can be misleading on imbalanced datasets.

TP + TN / (TP + TN + FP + FN)
Precision

Of all positive predictions, how many were correct. High precision = few false alarms.

TP / (TP + FP)
Recall (Sensitivity)

Of all actual positives, how many were detected. Critical in medical diagnosis.

TP / (TP + FN)
F1 Score

Harmonic mean of precision and recall. Balances both concerns.

2 * (P * R) / (P + R)
AUC-ROC

Area Under the Receiver Operating Characteristic Curve. Threshold-independent measure of discriminative ability.

Area under TPR vs FPR curve
Confusion Matrix

Table showing TP, TN, FP, FN counts. Reveals specific failure modes of each model.

Limitations
  • Tabular datasets are small (195-768 samples), leading to high variance in metrics.
  • No external validation on independent hospital datasets.
  • Transfer learning performance depends on similarity between ImageNet and medical domains.
  • Class imbalance in some datasets (e.g., Pneumonia) may bias results toward majority class.
  • Models trained on specific imaging equipment/protocols may not generalize to other clinical settings.
